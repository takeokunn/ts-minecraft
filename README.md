# ts-minecraft

[Minecraft](https://www.minecraft.net/) clone written by [Three.js](https://threejs.org/) and [TypeScript](https://www.typescriptlang.org/).

https://minecraft.takeokunn.org/

## Requirements

- NodeJS 18.x

## Install

```console
$ npm install
$ npm start # http://localhost:8080/
```
